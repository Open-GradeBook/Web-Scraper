Search Tips:
  • To view ALL courses in the selected term, enter an asterisk in the Room Number search field.

  • To view all courses with a particular prefix in the selected term, enter the course prefix in the Course ID search field. Examples: MA in the Course ID search field will return all courses and sections with the MA prefix; MA111 will return all sections of only MA111; MA2 will return all 200-level MA courses.

  • To view multiple course prefixes in the selected term, using the Course ID search field, enter desired course prefixes, separated by a pipe, such as MA|PH. Or, search for ALL HSSA classes by selecting from the Department list.

  • When searching by ID/Username: Username for students is usually the first six characters of the last name followed by first initial and middle initial; Username for faculty is usually the last name. Searching by ID requires entering an exact match of the Banner ID (8xxxxxxxx)

Additional Information:

The Class Schedule Options button will take you to the course matrix view page for the latest term for which the Registrar has released course data. The course matrix is a report of all permutations of course schedule options. This is a great advising tool. See the Class Schedule Options page for more details.

A student/faculty username link will show you the courses they are taking/teaching.

A course link will take you to a course roster page.

On the course roster page, the [Set Grid] link will produce a grid display of the cumulative schedules for the selected set of faculty, students or courses.

Partial match is available for Student Username, Faculty Username, Room and Course.
